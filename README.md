# D-D-Character-Chat
чат с персонажем на основе LLM  Загрузите своего персонажа D&amp;D, и нейросеть будет общаться от его лица. Проект позволяет создавать, редактировать и удалять персонажей, задавать их характеристики и вести с ними диалог в реальном времени. Стек: Python, FastAPI, JSON, Llama.cpp / Ollama. Промпты генерируются из JSON-описания персонажа.
